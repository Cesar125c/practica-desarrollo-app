# 🎭 Rick and Morty - API Explorer

Aplicación web para explorar personajes de Rick and Morty con sistema de login.

## 🚀 Características

- 🔐 Login simple (usuario: `admin`, contraseña: `12345678`)
- 🔍 Búsqueda en tiempo real de personajes
- 📱 Diseño responsive
- 🖼️ Vista detallada de cada personaje con episodios

## 📦 Instalación y ejecución

### Opción 1: Live Server (RECOMENDADA)

1. Instala la extensión "Live Server" en VS Code
2. Haz clic derecho en `index.html` → "Open with Live Server"

### Opción 2: Doble clic

1. Haz doble clic en `index.html`
2. El proyecto se abrirá en tu navegador

### Opción 3: Node.js (servidor profesional)

```bash
npm install   # Si tienes package.json
npm start
# o
node server.js
